# CIA Triad

## Objective
Understand the core principles of information security using the CIA Triad.

## CIA Triad Components

### Confidentiality
Confidentiality ensures that sensitive information is accessible only to authorized users.

Examples:
- passwords
- encryption
- access controls

### Integrity
Integrity ensures that data remains accurate and is not modified without authorization.

Examples:
- hash verification
- checksums
- file integrity monitoring

### Availability
Availability ensures that systems, applications, and data are accessible when needed.

Examples:
- backups
- redundancy
- disaster recovery

## Cybersecurity Relevance
The CIA Triad helps organizations design security controls to protect systems, data, and services from cyber threats.

## Key Learning
Effective cybersecurity requires balancing confidentiality, integrity, and availability to reduce security risks and maintain organizational operations.
