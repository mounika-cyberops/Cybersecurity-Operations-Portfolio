# Hash Functions and Checksums

## Objective
Understand the difference between checksums and hash functions in cybersecurity.

## Checksum
A checksum is used to detect accidental data corruption or errors during file transfer or storage.

## Hash Function
A hash function converts data into a fixed-length hash value used for integrity verification and security purposes.

## Common Hash Algorithms
- MD5
- SHA-1
- SHA-256

## Cybersecurity Relevance
Security teams use hash values to:
- verify file integrity
- identify known malicious files
- detect file tampering
- support malware investigations

## Key Learning
Checksums mainly help detect accidental changes, while hash functions are used for both integrity verification and security-related analysis.
