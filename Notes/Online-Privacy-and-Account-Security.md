# Online Privacy and Account Security

## Objective
Understand how online privacy, authentication mechanisms, secure browsing, and security awareness practices help protect users and organizations from cyber threats.
---

# Two-Factor Authentication (2FA)

## Overview
Two-factor authentication adds an additional verification layer beyond usernames and passwords.

## Common Authentication Factors
- passwords or PINs
- mobile devices or hardware tokens
- biometric verification
- verification codes

## Security Benefits
- reduces unauthorized access
- protects compromised passwords
- improves account security

## Limitations
2FA can still be targeted through:
- phishing attacks
- malware infections
- social engineering
---

# OAuth Authentication

## Overview
OAuth allows users to access third-party applications without directly sharing passwords.

## Security Benefits
- reduces password exposure
- improves login convenience
- supports secure delegated access

## Risks
- malicious third-party applications
- excessive application permissions
- compromised linked accounts
---

# Social Media Privacy Risks

## Risks of Oversharing
Attackers may collect publicly shared information for:
- phishing attacks
- identity theft
- social engineering
- credential guessing

## Sensitive Information to Avoid Sharing
- personal addresses
- travel plans
- financial information
- organizational details
- confidential business information
---

# Email Security and Spoofing

## Overview
Spoofed emails impersonate trusted senders to deceive users.

## Common Risks
- phishing attacks
- credential theft
- malware delivery
- business email compromise

## Security Measures
- verify sender addresses
- avoid suspicious links
- inspect attachments carefully
- report phishing emails
---

# Private Browsing and Web Privacy

## Overview
Private browsing modes reduce local storage of browsing history and cookies.

## Browser Privacy Modes
- Chrome Incognito
- Firefox Private Browsing
- Edge InPrivate
- Safari Private Browsing

## Limitations
Private browsing does not:
- hide activity from ISPs
- prevent network monitoring
- stop advanced tracking methods
---

# Public Wi-Fi Security

## Risks
Public wireless networks may expose users to:
- traffic interception
- man-in-the-middle attacks
- credential theft
- session hijacking

## Security Measures
- use VPN services
- avoid sensitive transactions
- disable file sharing
- verify trusted connections
---

# Password Managers

## Overview
Password managers securely store and generate strong passwords.

## Security Benefits
- reduces password reuse
- improves password complexity
- supports secure credential storage
- simplifies account management

## Security Recommendations
- use strong master passwords
- enable multi-factor authentication
- use trusted password manager solutions
---

# Safe Software Practices

## Risks of Untrusted Software
Installing software from untrusted sources may introduce:
- malware
- spyware
- trojans
- backdoors

## Best Practices
- download software from trusted vendors
- verify application legitimacy
- involve IT/security teams when required
---

# Phishing Awareness

## Common Indicators
- suspicious sender addresses
- urgent language
- unexpected attachments
- fake login pages
- spelling or grammar issues

## Security Measures
- verify links before clicking
- report suspicious emails
- avoid downloading unknown files
- conduct user awareness training
---

# SOC Relevance

SOC analysts investigate phishing attempts, suspicious logins, malware alerts, unauthorized account activity, VPN usage anomalies, and social engineering incidents to protect organizational assets.
---

# Key Learning

Strong authentication, privacy awareness, secure browsing habits, phishing detection, and responsible online behavior significantly improve cybersecurity protection for individuals and organizations.
