# Defensive Security Introduction

## Objective
Understand the role of defensive security and how security teams monitor and respond to cyber threats.

## Key Understanding
Defensive security focuses on protecting systems, networks, and organizational assets by detecting, investigating, and responding to suspicious activity and cyber attacks.

## Topics Learned
- Defensive security
- Security monitoring
- Incident response
- Threat detection
- Containment
- SOC analyst responsibilities

## Security Observation
In defensive security, the immediate priority during an attack is containment. Containment helps reduce the impact of attacks by stopping or limiting malicious activity before further damage occurs.

## Example Scenario
Multiple failed login attempts from a single IP address may indicate suspicious authentication activity or a brute-force attack attempt.

## What A SOC Analyst May Investigate
- Source IP address
- Authentication logs
- Failed login events
- Successful logins after failures
- Impacted user accounts

## Key Learning Outcome
SOC analysts monitor alerts and investigate suspicious behavior to help protect organizational systems and reduce security risks.
