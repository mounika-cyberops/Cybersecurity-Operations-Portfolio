
## Objective
Understand essential security practices used to protect devices, networks, accounts, and sensitive organizational data.
---

# Device Security

## Security Measures
- enable firewalls
- install antivirus and antispyware
- apply operating system updates
- install security patches
- encrypt sensitive data
- enable password protection

## Security Risks
- malware infection
- unauthorized access
- data theft
- ransomware attacks

## SOC Relevance
Security teams monitor endpoint alerts, antivirus detections, suspicious processes, and unauthorized access attempts.
---

# Firewall Protection

## Overview
Firewalls help prevent unauthorized access by monitoring and filtering incoming and outgoing network traffic.

## Benefits
- blocks malicious traffic
- reduces attack surface
- protects internal systems
- improves network security
---

# Antivirus and Antispyware

## Overview
Antivirus and antispyware solutions help detect, block, and remove malicious software.

## Common Threats
- viruses
- spyware
- trojans
- ransomware

## Security Practices
- regular scanning
- signature updates
- real-time protection
---

# Operating System and Browser Security

## Best Practices
- install security updates regularly
- patch known vulnerabilities
- remove unsupported software
- maintain secure browser settings

## Risks of Unpatched Systems
- exploit execution
- malware infections
- unauthorized access
---

# Password and Passphrase Security

## Strong Password Practices
- use long passwords or passphrases
- avoid dictionary words
- avoid password reuse
- use special characters
- enable multi-factor authentication

## Common Risks
- brute-force attacks
- password spraying
- credential theft
- unauthorized account access

## Security Recommendations
- use password managers
- avoid storing passwords insecurely
- monitor suspicious login activity
---

# Wireless Network Security

## Best Practices
- change default router credentials
- enable WPA2 or WPA3 encryption
- update router firmware
- disable unused wireless features

## Risks
- unauthorized network access
- traffic interception
- credential theft
---

# Public Wi-Fi Risks

## Common Threats
- eavesdropping
- man-in-the-middle attacks
- session hijacking
- malicious hotspots

## Security Measures
- use VPN services
- avoid sensitive transactions
- disable file sharing
- verify trusted networks
---

# IoT Device Security

## Risks
IoT devices may contain outdated software and weak security configurations that attackers can exploit.

## Security Measures
- isolate IoT devices on separate networks
- update firmware regularly
- disable unnecessary services
- change default passwords
---

# VPN Security

## Overview
Virtual Private Networks (VPNs) encrypt network traffic and help protect data during transmission.

## Benefits
- encrypted communication
- safer remote access
- protection on public Wi-Fi
- reduced interception risks
---

# SOC Relevance

SOC analysts monitor endpoint alerts, authentication activity, firewall logs, suspicious network behavior, malware detections, and unauthorized access attempts to improve organizational security.
---

# Key Learning

Strong device security, secure network configurations, password protection, regular updates, and user awareness significantly reduce cybersecurity risks and improve defensive security posture.
