## Objective
Understand common cyber attack infiltration methods, attacker techniques, and the importance of monitoring suspicious activities in security operations.

---

# Social Engineering

## Overview
Social engineering attacks manipulate users into revealing confidential information or performing unsafe actions.

## Common Techniques
- pretexting
- baiting
- tailgating
- phishing
- impersonation

## Possible Risks
- credential theft
- unauthorized access
- malware infection
- data exposure

## Security Measures
- security awareness training
- identity verification
- multi-factor authentication
- restricted physical access
  
# Denial-of-Service (DoS)

## Overview
A Denial-of-Service attack attempts to overwhelm systems, applications, or networks to disrupt normal operations.

## Common Impact
- service outages
- slow system performance
- network congestion
- operational disruption

## Security Measures
- firewall filtering
- traffic monitoring
- rate limiting
- intrusion prevention systems

# Distributed Denial-of-Service (DDoS)

## Overview
A DDoS attack uses multiple compromised systems (botnets) to flood a target with malicious traffic.

## Possible Indicators
- sudden traffic spikes
- abnormal network activity
- application unavailability

## Security Measures
- DDoS protection services
- traffic filtering
- network monitoring
- load balancing

# Botnets

## Overview
A botnet is a network of infected devices controlled remotely by attackers.

## Common Uses
- DDoS attacks
- malware distribution
- spam campaigns
- brute-force attacks

## SOC Relevance
Security teams monitor outbound traffic and command-and-control communication patterns to identify infected devices.

# On-Path (Man-in-the-Middle) Attacks

## Overview
Attackers intercept or modify communications between users and systems to steal sensitive information.

## Possible Risks
- credential theft
- session hijacking
- financial fraud
- sensitive data exposure

## Security Measures
- encrypted communication (HTTPS)
- VPN usage
- multi-factor authentication
- secure Wi-Fi practices

# SEO Poisoning

## Overview
Attackers manipulate search engine rankings to direct users to malicious websites.

## Possible Risks
- malware downloads
- phishing attacks
- credential compromise

## Security Measures
- verify website URLs
- web filtering
- user awareness training

# Password Attacks

## Common Types
- brute-force attacks
- dictionary attacks
- password spraying
- rainbow table attacks
- credential interception

## Possible Risks
- account compromise
- unauthorized access
- privilege escalation

## Security Measures
- strong password policies
- multi-factor authentication
- account lockout policies
- authentication monitoring

# Advanced Persistent Threats (APT)

## Overview
Advanced Persistent Threats are long-term, stealthy attacks targeting organizations or governments.

## Attacker Goals
- espionage
- persistent access
- sensitive data theft
- operational disruption

## SOC Relevance
SOC analysts monitor:
- abnormal behavior
- lateral movement
- persistence mechanisms
- suspicious privileged activity

# Key Learning

Cyber attackers use multiple infiltration techniques including social engineering, malware, password attacks, and network-based attacks to gain unauthorized access to systems and data.

Continuous monitoring, strong authentication, user awareness, and proactive security controls help organizations reduce cyber risks and improve incident detection capabilities.
